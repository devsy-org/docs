## Devsy Desktop

Devsy desktop automatically checks for updates on GitHub every 15minutes.
When an update will be available this popup will appear to confirm the update process:

<figure>
  <img src="/docs/media/devsy-update-available.png" alt="Devsy Flow" />
  <figcaption>Create Workspace</figcaption>
</figure>

The update will continue in background. When it's done you'll be prompted to restart the application.

<figure>
  <img src="/docs/media/devsy-update-done.png" alt="Devsy Flow" />
  <figcaption>Create Workspace</figcaption>
</figure>

At restart the application will be at latest version.

## Devsy Desktop - alternative

If you installed Devsy using deb or rpm packages, you'll need to [download the
new package](https://github.com/devsy-org/devsy/releases/latest/), and install it using your package manager again.

## Optional: Update Devsy CLI

If you just installed the CLI, you just need to download the latest version again
to update.
